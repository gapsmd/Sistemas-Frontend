¿Por qué estudiar Ingeniería en Sistemas Computacionales?

Afronta el mundo actual y los conocimientos para forjar el futuro, permitiendo el desarrollo y la creación de nuevas aplicaciones o sistemas que puedan ser de ayuda para una empresa y la sociedad en general.

Tópicos de la Carrera

Desarrollo de Software

Desarrolla programas a través de diferentes lenguajes de programación.

Ingeniería de Software

Aplica metodologías agiles para el desarrollo eficiente de proyectos de software.

Administración de Bases de Datos

Gestiona y almacena el flujo de datos de información de un sistema informático.

Programación web

Crea y desarrolla aplicaciones y sitios web para brindar soluciones a la empresas.

Redes y Telecomunicaciones

Permite la interconexión entre distintos equipos de cómputo de forma física o virtual.

Inteligencia Artificial

Aprende los fundamentos sobre sistemas informaticos de última generación.
